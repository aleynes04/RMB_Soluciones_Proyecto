# Prompt para Requerimientos

Analiza el planteamiento de nuestro proyecto de servicio técnico y ayúdame a redactar requerimientos funcionales claros, verificables y sin ambigüedad. Organízalos con identificadores RF-001, RF-002, etc. Para cada uno incluye: descripción, actor que lo ejecuta, entrada de datos, proceso esperado de negocio, salida (resultado) y nivel de prioridad.
