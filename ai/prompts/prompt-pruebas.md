# Prompt para Pruebas

Con base en los requerimientos funcionales, casos de uso y reglas de negocio de RMB Soluciones, genera casos de prueba funcionales y de integración. Para cada caso incluye: objetivo de verificación, precondiciones necesarias, pasos secuenciales, datos de prueba concretos (inputs), resultado esperado en el sistema y el criterio formal de aprobación (Pass/Fail).
