# Prompt para Casos de Uso

A partir de los requerimientos funcionales del sistema de servicio técnico y del perfil de los usuarios (técnicos y clientes), genera casos de uso estructurados. Para cada caso de uso incluye: identificador (CU-xxx), nombre, objetivo del actor, actor principal, precondiciones, flujo principal paso a paso, flujos alternos, excepciones (mensajes de error) y postcondiciones del sistema.
